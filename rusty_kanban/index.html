    How to Use the Kanban Board

    • Add tasks using the input field above
    • Click a task to select it
    • Shift + Click to select multiple tasks in a range
    • Ctrl/Cmd + Click to toggle task selection
    • Right-click a task to move it to another column
    • Click a task to edit its text
    • Press Enter or click outside to save changes
